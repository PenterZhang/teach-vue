# 第一天

## 准备工作

> 下载[node-js](https://nodejs.org/en/download)

### 安装完成后检查是否安装好，在命令窗口中输入以下命令：

```
    node -v

    # and

    npm -v
```
![node安装完成](./Images/node.png)

## 安装vue脚手架和VScode代码编辑器


> 安装vue脚手架开发环境详细参阅[vue-cli](https://cli.vuejs.org)

```
    npm install -g @vue/cli
    # OR
    yarn global add @vue/cli
```
